A simple web application that displays random Indian Quotes, Recipes, and Festivals.

Built using Node.js, Express.js, EJS, and serves a clean UI with dynamic content on each request.

Features

 Quote of the Day – Shows a random inspirational quote from Indian leaders.

 Recipe Section – Displays a random traditional Indian food recipe with ingredients & steps.

Festival Section – Shows a random Indian festival with date and description.

Clean UI with EJS templating & static assets.

Fast and lightweight Express server.

Tech Stack

Node.js

Express.js

EJS (Embedded JavaScript Templates)
